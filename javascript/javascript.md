1. 자바스크립트 언어가 다른 언어와 다른 특징을 3가지 서술하세요
- 동적 타입(dynamic type) : 변수 타입이 고정이 아니라, 실행 중에 바뀔 수 있습니다
- 단일 스레드 + 이벤트 루프 : 한 번에 하나의 작업만 처리하지만, 이벤트 루프와 큐로 비동기를 처리합니다
- 프로토타입 기반 객체지향 : 클래스 대신(ES6 전) 프로토타입 체인으로 상속을 구현할 수 있습니다

2. 변수에 대해 설명하고 let, const의 차이점을 서술하세요
- 변수: 값을 저장하기 위한 이름이 있는 메모리 공간
- let : 블록 스코프, 재할당 가능
- const : 블록 스코프, 재할당 불가 (단, 객체/배열 내부 값 수정은 가능)

3. 호이스팅 개념에 대해 설명해주세요
- 변수/함수 선언이 스코프의 최상단으로 끌어올려진 것처럼 동작하는 현상
- var, 함수 선언문, let, const 모두 호이스팅 대상이지만 var는 undefined로 초기화됨
- let, const는 초기화 전까지 일시적 사각지대(TDZ) 에 있어 접근 시 에러

4. 클로저 개념에 대해 설명해주세요
- 함수가 선언될 때의 외부 변수 환경을 기억해서, 나중에 호출되더라도 그 변수에 접근할 수 있는 기능합니다
- “내부 함수 + 그 함수가 선언될 당시의 외부 변수 묶음”이라고 이해하면 쉽습니다
